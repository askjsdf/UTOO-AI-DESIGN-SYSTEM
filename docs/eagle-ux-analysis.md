## 一、整体产品定位

Eagle 是一款面向设计师、创意工作者的**本地视觉资产管理工具**（Desktop App，macOS + Windows）。核心价值主张：

- **不订阅，买断制**（$29.95，终身 2 台授权，30 天免费试用）
- **支持 81+ 格式**（图片/视频/音频/字体/PDF/3D/源文件）
- **本地存储，速度极快**（关键词搜索 < 0.5 秒）
- **Eagle 5.0**（2026 Q1 公测）引入 AI Search + AI Action + MCP 支持

## 三、左侧导航栏详细功能

### 3.1 固定导航项（系统级，不可删除）

| 导航项 | 功能描述 |
|--------|----------|
| 全部（All） | 显示库中所有文件，不分文件夹 |
| 最近添加（Recent） | 按导入时间倒序，快速找到刚入库的文件 |
| 未分类（Uncategorized） | 显示尚未放入任何文件夹的孤立文件 |
| 已加星标（Starred） | 快速访问有星级评分的文件 |
| 颜色筛选（Color Filter） | 7 种主色调条形图，点击按颜色过滤全库 |
| 所有标签（All Tags） | 标签列表视图，显示每个标签下的文件数量 |

### 3.2 文件夹树

- **无限层级嵌套**：文件夹 > 子文件夹，无深度限制
- **展开/收起**：左侧箭头图标，支持键盘操作
- **自定义图标 + 颜色**：每个文件夹可设置独立图标和颜色，快速视觉识别
- **文件夹标签（Folder Tags）**：给文件夹赋标签后，进入该文件夹的所有文件自动被打上同样标签
- **多文件夹归属**：同一文件可同时属于多个文件夹（Alt + 拖拽），不占用额外磁盘空间（软链接机制）
- **拖拽排序**：文件夹可自由拖拽排列顺序
- **右键菜单**：新建子文件夹 / 重命名 / 删除 / 设置图标颜色 / 自动标签 / 移动到…

### 3.3 智能文件夹（Smart Folders）

**本质**：带规则的虚拟文件夹，内容由规则自动维护，不实际移动文件

**可用过滤条件**（可组合）：
- 文件名包含关键词
- 标签包含/不包含
- 颜色（主色调）
- 文件格式（jpg/png/mp4 等）
- 评分（1-5 星）
- 形状（横向/纵向/正方形/全景）
- 文件大小范围
- 导入日期范围

**逻辑运算**：多条件支持 AND 组合（"满足所有条件"），某些版本支持 OR

**使用场景举例**：
```
智能文件夹"高质量横图" = 评分 >= 4 AND 形状 = 横向 AND 格式 = PNG
智能文件夹"未标注"    = 标签为空 AND 评分为空
```

## 四、主内容区（文件网格）

### 4.1 视图模式

| 模式 | 说明 |
|------|------|
| 等比网格（Justified Grid） | 每行图片等高，宽度自适应，保留原始比例 — 默认模式 |
| 固定网格（Standard Grid） | 所有缩略图同尺寸方格，裁剪显示 |
| 列表视图（List View） | 单行展示，可显示/隐藏列（尺寸/格式/大小/日期/评分/标签） |

### 4.2 缩略图大小

- 拖动滑条（底部工具栏），实时调整缩略图尺寸
- 范围：极小（约 80px） ↔ 极大（约 400px）

### 4.3 排序选项

| 排序字段 | 支持方向 |
|----------|----------|
| 导入时间 | 升/降 |
| 文件名 | A-Z / Z-A |
| 文件大小 | 大→小 / 小→大 |
| 文件格式 | 按扩展名 |
| 评分 | 高→低 / 低→高 |
| 修改时间 | 升/降 |

### 4.4 背景色切换

三种主内容区背景：**纯黑 / 深灰 / 浅灰**，切换按钮在顶部工具栏

### 4.5 网格内信息显示

每个缩略图下方可切换显示：
- 文件名
- 文件格式 Badge
- 文件尺寸（px）
- 评分星星
- 颜色色条（5色调色板缩略版）

可全局开关"显示文件信息"

### 4.6 鼠标悬停行为

- 悬停缩略图右下角 `+` 图标 → 快速预览放大
- 悬停视频/GIF → 自动播放预览
- 悬停右下角 → 显示"加入文件夹"快捷按钮

### 4.7 多选行为

- 单击：选中单个文件
- Cmd/Ctrl + 单击：追加选中
- Shift + 单击：范围选中
- Cmd/Ctrl + A：全选
- 选中后：拖入文件夹 / 批量打标签 / 批量评分 / 批量导出 / 批量删除

### 4.8 文件右键菜单

- 打开（默认应用）
- 在新窗口打开（可钉到桌面前台）
- 加入文件夹（F 键快捷打开文件夹菜单）
- 复制到文件夹 / 移动到文件夹
- 添加标签
- 设置评分（1-5）
- 复制标签（从其他文件粘贴）
- 重命名
- 查找重复
- 反向图片搜索
- 导出
- 在 Finder/Explorer 中显示
- 删除

## 五、右侧检视器面板（Inspector）

**触发方式**：选中任意文件后，右侧面板自动显示；可通过快捷键或按钮折叠

### 5.1 检视器顶部

- **大预览缩略图**（点击进入全屏预览/详情模式）
- **文件名**（可点击编辑）

### 5.2 元数据字段

| 字段 | 说明 | 可编辑？ |
|------|------|---------|
| 星级评分 | 1-5 星，点击星星设置；键盘 1-5 | 是 |
| 标签（Tags） | 多标签输入，逗号分隔，自动补全建议 | 是 |
| 备注/描述（Note） | 多行文本，全文可搜索 | 是 |
| 来源 URL | 浏览器扩展导入时自动填入 | 是 |
| 颜色调色板 | 自动提取的 5-8 主色，显示色块；右键色块→按此颜色搜索 | 否（自动） |
| 文件格式 | 扩展名 | 否 |
| 文件尺寸 | 宽 × 高（px），容量（KB/MB） | 否 |
| 形状分类 | 横向/纵向/正方形/全景（自动） | 否 |
| 导入时间 | 首次导入的时间戳 | 否 |
| 修改时间 | 文件最后修改时间 | 否 |
| 文件路径 | 本地磁盘路径（Eagle 复制了文件） | 否 |

### 5.3 颜色调色板详细行为

- 图片导入时自动提取 **5~8 个主色调**
- 每色块显示大致 HEX / RGB 值（悬停）
- **右键色块** → "搜索相似颜色" → 过滤整库
- 颜色信息生成算法在 Eagle 4 Build 12 后明显提速（大库下尤其明显）

### 5.4 评论/批注（Comments）

- 视频文件支持**带时间戳的评论**（标注某帧的问题/灵感）
- 普通图片支持多条评论，每条有创建时间
- Comment API 完整 CRUD，可被插件扩展

## 六、搜索与过滤系统

### 6.1 全局搜索框（顶部）

- 搜索范围：**文件名 + 标签 + 备注 + URL + 文件夹名 + 文件夹描述 + 格式**
- 搜索速度：< 0.5 秒（本地索引）
- 支持 OR 运算符和括号：`(blue OR red) minimal`，`(logo || brand) 2024`
- 模糊搜索：部分匹配即可命中

### 6.2 过滤器面板（搜索结果/文件夹内）

过滤器可叠加（AND 逻辑），包括：

| 维度 | 可选值 |
|------|--------|
| 颜色 | 7 色调色板点选（红/橙/黄/绿/青/蓝/紫 + 黑白灰） |
| 形状 | 横向 / 纵向 / 正方形 / 全景 |
| 评分 | ★☆☆☆☆ 以上 / ★★☆☆☆ 以上 / … / 仅 5 星 |
| 文件格式 | jpg / png / gif / mp4 / pdf / ai / psd / svg / … |
| 标签 | 多标签 AND 过滤 |
| 文件大小 | 自定义范围（KB/MB） |
| 导入时间 | 今天 / 本周 / 本月 / 自定义日期范围 |
| 修改时间 | 同上 |
| 时长 | 视频/音频专用，按秒范围筛选 |
| 尺寸 | 宽/高像素范围 |

### 6.3 随机浏览模式（Random）

- 按钮触发：随机打乱当前文件夹内文件顺序
- 用于灵感探索，避免视觉疲劳导致忽略老资产

### 6.4 反向图片搜索

- 选中图片 → 右键 → "反向图片搜索"
- 跳转到 Google Images / TinEye 查找来源

## 九、预览与详情模式

### 9.1 轻量预览（Hover/Spacebar）

- 鼠标悬停 + 图标 或 按 Z → 当前位置浮层放大预览
- 鼠标移走即关闭
- 不影响当前视图状态

### 9.2 详情模式（Detail Mode）

- 双击缩略图 或 按 C → 进入全屏/大图模式
- 支持左右键/鼠标滑动切换图片
- 底部显示完整 Inspector 信息
- 支持缩放（Cmd+=/-, Ctrl+9 重置，反引号适配窗口）
- 图片可在预览区域内拖动重定位

### 9.3 多窗口钉屏

- 右键 → "在新窗口打开" → 独立预览窗口
- 窗口可钉在桌面前台（Stay on Top），边设计边参考

### 9.4 视频/GIF 预览

- GIF：悬停自动播放；C 键进入帧查看模式
- 视频：悬停显示进度条，可拖拽预览任意帧；支持设置某帧为封面缩略图
- 播放控制：速度调节（0.25x / 0.5x / 1x / 2x），循环开关
- 带时间戳评论系统

### 9.5 音频预览

- 悬停播放
- 列表/网格中显示波形图或专辑封面缩略图

## 十五、UX 关键细节总结（用于复刻对标）

### 15.1 必须实现的核心 UX 模式

1. **左侧导航"固定项"不可缺**：全部 / 最近添加 / 未分类 — 这是用户每天的起点
2. **智能文件夹是核心生产力**：一旦设置好规则，完全不需要手动归档
3. **Inspector 颜色调色板必须可点击**：右键色块→按颜色搜索是高频操作
4. **鼠标悬停自动播放 GIF/视频**：不需要点击，减少摩擦
5. **等比网格（Justified Grid）**是默认视图，不是固定方格
6. **缩略图大小滑条**是用户非常依赖的功能（密度调节）
7. **F 键打开归类菜单** + **Shift+D 重复上次** — 批量归档效率关键
8. **标签自动补全**：输入时显示已有标签建议，防止标签碎片化
9. **导入重复检测**：必须在导入流程中内置，不能事后扫描
10. **自定义文件夹图标+颜色**：视觉辨识度，大库必需

### 15.2 可暂缓实现的功能（UI 上标注"即将上线"）

## 十六、建议实现优先级（下一阶段 Roadmap）

### P0 — 本周实现（核心体验）

1. **等比网格（Justified Grid）**：CSS `masonry` 或 JS 计算行高，每行图片等高
2. **缩略图大小滑条**：底部 Range input，控制 `--thumb-size` CSS 变量
3. **左侧固定导航项**："全部" / "最近添加" / "未分类"
4. **颜色自动提取**：图片导入后用 canvas 取样，存入 `_meta.json` 的 `fileColors` 字段
5. **Inspector 颜色调色板点击搜索**

### P1 — 近期实现（效率功能）

6. **标签自动补全**：从 `_meta.json` 聚合所有已有标签，输入时建议
7. **星级评分持久化**：写入 `_meta.json` 的 `fileRating` 字段
8. **排序功能**：按名称/日期/大小/评分排序
9. **过滤器**：按格式 / 评分 / 标签过滤
10. **备注持久化**：写入 `_meta.json` 的 `fileNotes` 字段

### P2 — 中期实现（高级功能）

11. **智能文件夹**
12. **颜色过滤**（左侧导航颜色条）
13. **悬停预览放大**
14. **批量操作**（多选 + 批量标签/移动）
15. **文件夹图标/颜色自定义**
